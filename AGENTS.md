# AGENTS.md

## Starting a project

You always look for:
- Scope
- Implementation tasks
- Constraints
- Goals/Vision

If you don't have this information, you create/request it.

## How to approach a problem

When tacking a problem such as a bug report, you always follow this procedure:
1. Identify the root cause of the problem.
2. Propose a root-cause solution. Your solution always solves the general case.
3. Implement said root-cause solution.
